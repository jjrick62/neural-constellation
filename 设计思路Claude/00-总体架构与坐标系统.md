# 00 — 总体架构与坐标系统

## 坐标系统

```
世界坐标系：Y-up 右手系
原点 (0, 0, 0) = 圆柱底面圆心上方 2 单位处（留出底部入口空间）

圆柱参数：
  内壁半径 R_inner = 4.0
  外壁半径 R_outer = 4.15
  总高度   H_total = 24.0
  Y 范围   [-10, 14]
```

## 圆柱分区

圆柱沿 Y 轴分为 4 个功能区，每个区高度 6 单位：

| 区号 | 名称 | Y 范围 | 滚动进度 | 内容 |
|------|------|--------|----------|------|
| Z0 | 扉页 | [-10, -4] | 0% – 20% | VINCENT 文字悬浮于圆柱中轴 |
| Z1 | 项目 | [-4, 2] | 20% – 45% | 6 张项目卡片贴于内壁 |
| Z2 | 技能 | [2, 8] | 45% – 70% | 技能标签悬浮于中心空间 |
| Z3 | 关于 | [8, 14] | 70% – 100% | 金句 + 联系方式悬浮于中轴 |

区间之间有 1 单位的过渡缓冲区（blend zone），内容做交叉淡化。

## 圆柱内壁坐标系

```
内壁上任意点 P 用柱坐标 (r, θ, y) 表示：
  r = R_inner = 4.0（固定在内壁）
  θ ∈ [0, 2π)，从 +X 轴逆时针计量
  y ∈ [-10, 14]

内容贴于内壁时：法线方向指向圆心（即 -position.normalized()）
摄像机沿内壁移动时：位置在 r ≈ 3.2-3.8 处（距内壁 0.2-0.8 单位）
```

## 摄像机约束

```
1. 摄像机始终在圆柱内部：sqrt(cam.x² + cam.z²) < R_inner
2. 摄像机 Y 始终在 [-10, 14] 内
3. LookAt 目标始终在圆柱中轴 ±1 单位范围内
4. FOV 范围 [35°, 55°]
5. 近裁剪面 0.1，远裁剪面 50
```

## 滚动映射

```
页面总高度 = 400vh（CSS 设置 body height）
滚动位置 scrollY ∈ [0, maxScroll]
  其中 maxScroll = body.scrollHeight - window.innerHeight

归一化进度 p = clamp(scrollY / maxScroll, 0, 1)

映射函数 f(p) → 摄像机 Y 坐标：
  非均匀映射：内容密集区（Z1, Z2）分配更多滚动行程
  Z0: p ∈ [0, 0.20]    → y ∈ [-10, -4]   线性
  Z1: p ∈ [0.20, 0.45] → y ∈ [-4, 2]     线性（25% 行程覆盖 6 单位）
  Z2: p ∈ [0.45, 0.70] → y ∈ [2, 8]      线性（25% 行程覆盖 6 单位）
  Z3: p ∈ [0.70, 1.00] → y ∈ [8, 14]     线性（30% 行程覆盖 6 单位）

区间边界 1 单位用于过渡特效：
  p ∈ [0.19, 0.21]: Z0→Z1 切屏过渡
  p ∈ [0.44, 0.46]: Z1→Z2 切屏过渡
  p ∈ [0.69, 0.71]: Z2→Z3 切屏过渡
```
