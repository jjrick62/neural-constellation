# 05 — 切屏过渡特效

## 触发条件

```
当摄像机 Y 坐标穿过分区边界 ±0.5 单位范围时，触发切屏过渡：

  Z0→Z1 边界 y=-4：触发区间 [-4.5, -3.5]，持续时间对应滚动 2%
  Z1→Z2 边界 y=2：  触发区间 [1.5, 2.5]，持续时间对应滚动 2%
  Z2→Z3 边界 y=8：  触发区间 [7.5, 8.5]，持续时间对应滚动 2%
```

## 过渡特效规格

### 阶段 1：径向擦除（0 → 30% 过渡进度）

```
效果：从屏幕中心向外扩散的金色光环擦除
实现：
  - 一个全屏 Quad（OrthographicCamera 渲染），使用自定义 Shader
  - Shader 输入：uTransition（0→1 过渡进度）、uTime
  - 输出：
    float dist = length(gl_FragCoord.xy / resolution - 0.5) * 2.0;  // 0=中心, 1=角落
    float radius = uTransition * 1.5;  // 扩散半径
    float wipe = smoothstep(radius - 0.05, radius + 0.05, dist);
    // wipe=0 处显示当前帧，wipe=1 处显示下一帧的 hint

  - 擦除边缘带金色辉光：
    float edge = 1.0 - abs(dist - radius) * 20.0;
    edge = clamp(edge, 0.0, 1.0);
    vec3 glowColor = vec3(0.83, 0.66, 0.26) * edge * 0.6;
```

### 阶段 2：粒子爆发（20% → 70% 过渡进度）

```
效果：擦除边缘向外喷出金色粒子
实现：
  - 在边界 Y 高度生成 500 个临时粒子
  - 初始位置：圆柱内壁上的环形分布（r=3.8, y=边界Y, θ 均匀）
  - 初始速度：沿径向向内 + 随机扰动
    velocity = normalize(-position_xy, random(-1,1)) × (2 + random × 3)
  - 生命周期：0.5 秒
  - 大小衰减：从 0.06 线性衰减到 0
  - 颜色：金色 #D4A843 → 透明
  - 渲染：单独的 Points 对象，AdditiveBlending
  - 过渡结束后自动 dispose
```

### 阶段 3：内容交叉淡化（50% → 100% 过渡进度）

```
效果：旧区内容和新内容做 opacity 交叉淡化
实现：
  旧区 Group 所有材质 opacity = 1 - t（t ∈ [0.5, 1.0] 映射到 [0, 1]）
  新区 Group 所有材质 opacity = t
  两组同时 visible=true 直到过渡完成
  过渡完成后旧区 Group visible=false
```

## 过渡时间线总览

```
过渡总时长：对应约 300ms 滚动时间（在 scrub: 0.5 下约 150ms 实际）

  0ms ───── 阶段1 径向擦除 ───── 90ms
  60ms ───── 阶段2 粒子爆发 ───── 210ms
  150ms ──── 阶段3 内容淡化 ───── 300ms

三个阶段有重叠，不是严格串行
```

## 实现架构

```
class ScreenTransition {
  constructor(renderer)
  // 管理过渡状态机和特效资源

  trigger(fromZone, toZone)
  // 触发一次过渡
  // fromZone, toZone ∈ {0,1,2,3}

  update(delta)
  // 更新过渡进度，驱动三个阶段
  // 返回 { isActive: bool, progress: 0-1 }

  render(scene, camera)
  // 渲染过渡特效到屏幕
  // 内部使用全屏 Quad + ShaderMaterial

  _spawnBurstParticles(boundaryY)
  // 在指定 Y 高度生成爆发粒子

  dispose()
}
```

## 过渡期间的场景管理

```
过渡期间（progress 0→1）：
  - 摄像机轨迹继续正常移动（不停顿）
  - 星空粒子正常渲染
  - 旧区和新区的 content Group 都保持 visible=true
  - 过渡 Quad 渲染在最上层（z-index 等效）
  - 过渡完成后旧区 visible=false，过渡 Quad 移除
```
