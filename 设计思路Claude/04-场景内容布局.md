## 内容可见性管理

```
不通过 scene.visible 切换场景（会导致粒子也切换）
而是每个场景的内容挂在独立的 Group 上：
  scene1Group, scene2Group, scene3Group, scene4Group

可见性规则：
  当前区 = Z0: scene1Group.visible = true,  其余 false
  当前区 = Z1: scene2Group.visible = true,  其余 false
  当前区 = Z2: scene3Group.visible = true,  其余 false
  当前区 = Z3: scene4Group.visible = true,  其余 false

过渡区（blend zone）：两个相邻 Group 同时 visible，各自的 content 做 opacity 交叉淡化
  例如 y∈[-4.5, -3.5]：scene1Group 和 scene2Group 都可见
    scene1Group 内容整体 opacity = 1 - blend_weight
    scene2Group 内容整体 opacity = blend_weight
```
