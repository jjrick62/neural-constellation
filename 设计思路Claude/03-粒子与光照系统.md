# 03 — 粒子与光照系统

## 粒子系统

### 总粒子数：120,000（四分区独立配置）

### 分区密度（每区粒子数独立控制，不共享权重池）

```js
const zoneDefs = [
  { yMin: -10, yMax: -4, count: 10000 },  // Z0 扉页 · 稀疏
  { yMin:  -4, yMax:  2, count: 34000 },  // Z1 项目 · 中等
  { yMin:   2, yMax:  8, count: 43000 },  // Z2 技能 · 最密
  { yMin:   8, yMax: 14, count: 33000 },  // Z3 关于 · 适中
];
```

### 粒子生成算法（柱坐标原生）

```
function generateParticles():
  idx = 0
  for each zone in zoneDefs:
    for j in 0..zone.count:
      1. Y: zone.yMin + random() × (zone.yMax - zone.yMin)  // 区内均匀
      2. θ: random() × 2π                                   // 均匀环绕
      3. r: R_MIN + (R_MAX - R_MIN) × √random()             // 偏壁面（外密内疏）
      4. (x, z) = (r·cosθ, r·sinθ)
      5. positions[idx] = (x, y, z)
      6. worldPosStore[idx] = (x, y, z)
      7. size = baseSize(r) × noise(x,y,z)
      8. idx++
```

约束：r ∈ [R_MIN=0.3, R_MAX=CYLINDER.R_INNER=4.0]，天生在圆柱内

### 粒子大小噪声函数

```
size_noise(pos) = baseSize × (
  1.0
  + sin(pos.x * 7.31 + pos.z * 3.17) * 0.12
  + cos(pos.y * 5.73 + pos.x * 2.91) * 0.10
  + sin((pos.x + pos.y) * 4.19 + pos.z * 6.53) * 0.08
)

baseSize 取决于粒子到最近聚类中心的距离 d：
  baseSize = clamp(0.08 - d × 0.006, 0.015, 0.12)
  （中心附近粒子较大，边缘较小）

最终 size = clamp(size_noise, 0.012, 0.15)
```

### 粒子 Shader 光照计算

```
// Vertex Shader
attribute float aSize;
attribute vec3 aWorldPos;       // 粒子的固定世界坐标（不受 group 旋转影响）
varying vec3 vWorldPos;
uniform float uPointScale;

void main() {
  vec4 mvPosition = modelViewMatrix * vec4(position, 1.0);
  gl_PointSize = aSize * uPointScale * (300.0 / -mvPosition.z);
  gl_PointSize = clamp(gl_PointSize, 0.5, 20.0);  // 限制最大最小
  gl_Position = projectionMatrix * mvPosition;
  vWorldPos = aWorldPos;
}

// Fragment Shader
varying vec3 vWorldPos;

uniform vec3 uLightPos[4];     // 最多 4 个动态光源位置
uniform vec3 uLightColor[4];   // 对应光源颜色
uniform int  uLightCount;      // 实际光源数
uniform vec3 uCameraPos;       // 摄像机世界坐标

void main() {
  // 圆形蒙版
  float d = length(gl_PointCoord - 0.5) * 2.0;
  if (d > 1.0) discard;
  float alpha = 1.0 - smoothstep(0.15, 0.85, d);

  // 光照累加
  vec3 totalLight = vec3(0.0);
  for (int i = 0; i < 4; i++) {
    if (i >= uLightCount) break;
    float dist = length(vWorldPos - uLightPos[i]);
    float att = 1.0 / (1.0 + dist * dist * 0.003);  // 平方反比衰减系数
    totalLight += uLightColor[i] * att;
  }

  // 摄像机距离色偏（大气透视）
  float camDist = length(vWorldPos - uCameraPos);
  float warmth = exp(-camDist * 0.025);
  vec3 nearColor = vec3(1.0, 0.85, 0.6);   // 暖金
  vec3 farColor  = vec3(0.35, 0.45, 0.65);  // 冷蓝
  vec3 baseColor = mix(farColor, nearColor, clamp(warmth, 0.0, 1.0));

  // 静态高亮噪声炫光（少数粒子产生闪烁反光）
  float sparkle = abs(
    sin(vWorldPos.x * 23.7 + vWorldPos.y * 17.3)
    + cos(vWorldPos.y * 31.1 + vWorldPos.z * 19.7)
    + sin(vWorldPos.z * 27.3 + vWorldPos.x * 21.1)
  );
  sparkle = pow(sparkle / 3.0, 4.0) * 0.5;

  // 时间闪烁：每粒子独立相位，sin 取高次幂使闪烁锐利（眨眼感非平滑渐变）
  float phase = vWorldPos.x * 13.37 + vWorldPos.y * 17.73 + vWorldPos.z * 11.11;
  float twinkle = 0.5 + 0.5 * sin(uTime * 2.5 + phase);
  twinkle = pow(twinkle, 6.0);
  float twinkleBoost = 0.6 + 0.4 * twinkle;  // 亮度在 60%~100% 间波动

  vec3 finalColor = baseColor * totalLight * (1.0 + sparkle) * twinkleBoost;
  float finalAlpha = alpha * (0.25 + length(totalLight) * 0.7) * twinkleBoost;

  gl_FragColor = vec4(finalColor, clamp(finalAlpha, 0.0, 1.0));
}
```

### 光源配置

```
场景中的 PointLight 和粒子 Shader 中的 uLightPos 保持同步：

Light 1（主暖光）：
  PointLight: color #D4A843, intensity 30, distance 30
  Shader: uLightPos[0], uLightColor[0] = (1.0, 0.83, 0.26)

Light 2（辅琥珀光）：
  PointLight: color #B8942E, intensity 15, distance 25
  Shader: uLightPos[1], uLightColor[1] = (0.72, 0.58, 0.18)

Light 3（补冷白光）：
  PointLight: color #FFF8E7, intensity 8, distance 20
  Shader: uLightPos[2], uLightColor[2] = (1.0, 0.97, 0.90)

AmbientLight: color #111122, intensity 1.0（全局环境光底子，避免纯黑）

光源位置随摄像机 Y 移动：
  Light 1 始终在摄像机前方 2-4 单位（cam_pos + (0, 0, 3) 旋转到摄像机朝向）
  Light 2 始终在摄像机下方 3 单位（补底光）
  Light 3 在圆柱中轴当前高度（(0, cam_y, 0)）
```

### 渲染设置

```
粒子材质：
  transparent: true
  depthWrite: false
  blending: THREE.NormalBlending   ← 关键：不用 AdditiveBlending，
                                      粒子被外部光源照亮后反光，
                                      不是自发光
  depthTest: true                  ← 允许粒子间遮挡
```
