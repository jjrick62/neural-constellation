# 02 — 摄像机运动系统

## 核心约束

```
1. 摄像机位置始终满足：1.0 < sqrt(x²+z²) < 3.9（距中轴 1-3.9 单位，即圆柱内）
2. 默认距离中轴（轨道半径）：r_cam = 3.5
3. LookAt 目标 Y = 摄像机 Y - (1~3)，即摄像机略微俯视
4. LookAt 目标 XZ 始终在圆柱中轴 ±1 单位内
5. FOV 范围 [38°, 52°]，默认 45°
```

## 逐区摄像机轨迹

### Z0 — 扉页区螺旋上升

```
起始帧（p=0.00, y=-8）：
  position: (0, -8, 3.5)        ← 正前方，距中轴 3.5
  lookAt:   (0, -6, 0)          ← 看向中央的 VINCENT 文字
  fov:      48

结束帧（p=0.19, y≈-4.1）：
  position: (3.0*sin(0.7π), -4.1, 3.0*cos(0.7π))
         ≈ (2.43, -4.1, 1.76)
  lookAt:   (0, -3, 0)          ← 看向 Z1 入口方向
  fov:      45

中间轨迹：position.xz 做 0.7π 弧度（约 126°）的顺时针弧线
  θ(t) = 0 + 0.7π × t  （t ∈ [0, 1]，zone 内归一化进度）
  x(t) = r_cam × sin(θ(t))，但 r_cam 从 3.5 渐变为 3.0
  z(t) = r_cam × cos(θ(t))
  y(t) = lerp(-8, -4, t)

lookAt.y(t) = lerp(-6, -3, t)  ← 随上升而抬高视线
```

### Z1 — 项目区扫视运动

```
此区不做完整绕柱，而是线性扫视连接卡片位置前方

定义 6 个子关键帧（对应 6 张卡片）：
  K1 (p=0.20, y=-4)：  position (0, -4, 3.5)，    看向 θ=60° 卡片
  K2 (p=0.24, y=-3)：  position (3.2, -3, 1.5)，   看向 θ=0° 卡片
  K3 (p=0.28, y=-2)：  position (1.2, -2, 3.3)，   看向 θ=300° 卡片
  K4 (p=0.33, y=-1)：  position (-1.8, -1, 3.0)，  看向 θ=240° 卡片
  K5 (p=0.38, y=0)：   position (-3.2, 0, 1.2)，   看向 θ=180° 卡片
  K6 (p=0.44, y=1.5)： position (0, 1.5, 3.5)，    看向 θ=120° 卡片

每个子帧之间用 Catmull-Rom 曲线插值（保证 C1 连续）
position 在子帧之间做平滑弧线，不经过圆柱中轴
lookAt 在每个子帧指向对应卡片位置（卡片位置见 01 文档）
```

### Z2 — 技能区轨道运动

```
起始帧（p=0.46, y=2.5）：
  position: (3.5*cos(0), 2.5, 3.5*sin(0)) = (3.5, 2.5, 0)
  lookAt:   (0, 5, 0)              ← 看向技能区中心
  fov:      42

结束帧（p=0.69, y≈7.5）：
  position: 绕行约 1.5 圈
  θ_end = 1.5 × 2π ≈ 3π (540°)
  position: (3.5*cos(3π), 7.5, 3.5*sin(3π)) = (-3.5, 7.5, 0)
  lookAt:   (0, 8, 0)              ← 跟随视线上升
  fov:      40

轨迹：圆柱内壁螺旋
  θ(t) = 0 + 3π × t，t ∈ [0, 1]
  r(t) = 3.5（恒定轨道半径）
  y(t) = lerp(2.5, 7.5, t)
  lookAt.y(t) = lerp(5, 8, t)

FOV 从 42 渐缩到 40（望远感随高度增强）
```

### Z3 — 关于区拉远

```
起始帧（p=0.71, y=8.5）：
  position: (0, 8.5, 3.5)
  lookAt:   (0, 11, 0)
  fov:      48

结束帧（p=1.00, y=14）：
  position: (0, 13, 3.8)         ← 略微后撤
  lookAt:   (0, 12, 0)
  fov:      52                     ← 放宽视野

轨迹：纯垂直上升 + 轻微后撤
  x(t) = 0（不动）
  z(t) = lerp(3.5, 3.8, t)
  y(t) = lerp(8.5, 13.5, t)
```

## 区间过渡

```
区间之间的 1 单位 Y 作为 blend zone：
  Z0→Z1 (y∈[-4.5, -3.5])：摄像机轨迹和 lookAt 用 crossfade 混合
      blend_weight = clamp((y + 4.5) / 1.0, 0, 1)
      position = lerp(Z0_end_pos, Z1_start_pos, blend_weight)

  Z1→Z2 (y∈[1.5, 2.5])：同上
  Z2→Z3 (y∈[7.5, 8.5])：同上
```

## 插值器

```
区间内插值使用 easeInOutCubic：
  f(t) = t < 0.5 ? 4*t³ : 1 - pow(-2*t + 2, 3) / 2

区间过渡使用 easeInOutQuad（更柔）：
  f(t) = t < 0.5 ? 2*t² : 1 - pow(-2*t + 2, 2) / 2
```
