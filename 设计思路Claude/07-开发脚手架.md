# 07 — 开发脚手架

> 开发阶段专用的调试辅助系统，不参与最终产品渲染。

## 组件清单

### 1. 调试坐标轴 (DebugAxes)

```
原点：圆柱底面中心 (0, -10, 0)
六向轴长：±X, ±Y, ±Z 各 24 单位（双向，总跨 48 单位）
材质：MeshBasicMaterial { color: #ffffff, depthTest: false, depthWrite: false }
线型：CylinderGeometry(radius=0.06, height=axisLen, segments=8)

X 轴（±）：rotation.z = -π/2，沿 ±X 方向
Y 轴（±）：无旋转，沿 ±Y 方向
Z 轴（±）：rotation.x = π/2，沿 ±Z 方向

原点标记：SphereGeometry(radius=0.24, 16, 16)
六向端点标记：SphereGeometry(radius=0.15, 8, 8) × 6

刻度：
  - 每 4 单位一个主刻度（0.5 长竖线）+ 数字标签
  - 数字标签：Canvas 纹理 Sprite，白色 monospace 36px，scale(1.2, 0.6, 1)

depthTest: false 确保轴始终可见（不被粒子遮挡）
depthWrite: false 避免写入深度缓冲干扰其他渲染
```

### 2. 自由摄像机 (FreeCam)

```
触发：页面右上角按钮 (#cam-toggle) 切换 SCROLL / FREECAM

控制映射：
  W/S       — 沿视线方向 前进/后退
  A/D       — 垂直视线方向 左移/右移
  Q/E       — 左右旋转 (yaw)
  R         — 仰视 (pitch up)
  T         — 俯视 (pitch down)
  Space     — 世界 Y 轴上升
  Shift     — 世界 Y 轴下降
  鼠标滚轮   — 前进/后退（带惯性衰减，accumulate deltaY × 0.15 speed，残余 ×0.6）

移动速度：8 单位/秒
旋转速度：2.5 弧度/秒
pitch 限制：[-1.4, 1.4] rad (±80°)，防止万向节锁
摄像机范围：无约束（开发阶段可自由飞出圆柱）

模式切换逻辑：
  - 自由模式：body overflow-y=hidden 拦截滚动，WASD 生效
  - 滚动模式：body overflow-y=auto 恢复滚动，键盘输入忽略
  - 切换时按钮 blur() 防止 Space 二次触发
  - 自由模式显示底部提示面板 (#fly-hint)
  - 保存/恢复 scrollY 位置

底部提示面板：
  固定定位 bottom:24px，居中，半透明黑底+金色边框
  文本："W/S 进退 · A/D 平移 · Q/E 转头 · R 仰/T 俯 · Space/Shift 升降 · 滚轮进退"
  pointer-events: none 不阻挡点击
```
