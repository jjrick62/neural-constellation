# 01 — 圆柱容器与空间分区

## 圆柱体视觉呈现

### 不渲染完整圆柱 Mesh

圆柱本身不作为可见几何体渲染。内壁边界通过以下方式隐式表达：
- 内容元素（卡片、标签）的空间位置自然勾勒出圆柱轮廓
- 粒子系统的高斯聚类中心沿圆柱内壁分布，粒子密度在壁附近增强
- 摄像机运动受圆柱半径约束，视角始终在柱内

### 可选：内壁参考线

如需视觉提示，渲染以下元素（默认关闭，通过参数开启）：
```
- 8 条垂直细线沿内壁等距分布（θ = 0, π/4, π/2, ... 7π/4）
  每条线：y ∈ [-10, 14]，线宽 0.005，颜色 #333333
- 4 个水平细环在分区边界处（y = -4, 2, 8）
  每个环：r = 4.0，线宽 0.005，颜色 #444444
```

## 粒子密度分布函数

```
总体密度函数 D(x, y, z) = Σ(weight_i × gaussian(distance_to_cluster_i, spread_i))

其中 gaussian(d, σ) = exp(-d² / (2σ²))

粒子生成算法：
  for i in 0..count:
    1. 按 weight 加权随机选一个聚类中心 C_k
    2. 用 Box-Muller 生成三维高斯偏移 (ox, oy, oz)，各分量标准差 = spread_k
    3. 位置 = C_k + (ox, oy, oz)
    4. 如位置超出圆柱 (sqrt(x²+z²) < 1 或 > R_inner*1.3)，重新采样

每聚类中心的粒子配额 = count × weight_k
```
